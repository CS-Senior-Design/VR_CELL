EndoUI:

The prefab is called EndoUI.

Step1:

Attach endo control script directly to EndoUI object.

Step2:

Fill all global variables in Endo Control Script.

Step3:

Can use left and right buttons to move between panels without VR headset, otherwise us regular XRRay Interactors.


NOTES:

All the text for each panel is stored in private global string variables inside Endo Control Script.
